3. Extension Background

All hData section documents identified by an XML namespace must be declared in the extensions node of the root document. Extensions are either "mandatory" or "optional." Each section document has an address expressed as a Uniform Resource Identifier (URI). Each URI is registered in the extensions node of the root document. Sections can contain either single data type documents (e.g., XML, JSON, HTML) or URIs referencing other sections. Section document content must be expressed in XML and document types must be declared in the extensions node of the root document.

4. Description

4.1. Purpose

Allergy contains information about the allergies and intolerances associated with a patient.

4.2. Description

NameTypeMandatory/OptionalNotes
id string M Unique id for the allergy entry
code cd M

The coded value of the allergy represented

Ingredient: Food and non-medicinal allergies/Sensitivities SHALL be coded as specified in http://www.fda.gov/ForIndustry/DataStandards/StructuredProductLabeling/ucm162523.ht

Medication Class: Allergies/Drug Sensitivity to a class of medication Shall contain a value descending from the NDF-RT concept types of "Mechanism of Action - N0000000223", "Physiologic Effect - N0000009802" or "Chemical Structure - N0000000002". NUI will be used as the concept code. For more information, please see the Web Site http://www.cancer.gov/cancertopics/terminologyresources/page5

Medications: Allergies/Drug Sensitivity to a specific medication Shall contain RxNorm normal forms for concepts type of "Brand Name" or Brand Name Packs. The Brand name concepts can be found in the RxNORM file RXCONSO.RRF selecting all terms where SAB=RXNORM (selecting the normal forms), and TTY=BN (selecting the brand names) or TTY=BPCK (selecting the brand name packs)
status string O

Entry component: The status of the Allergy.

type SimpleCode M

This describes the type of product and intolerance suffered by the patient

Valid Values:

  • 420134006 -- Propensity to adverse reactions
  • 418038007 -- Propensity to adverse reactions to substance
  • 419511003 -- Propensity to adverse reactions to drug
  • 418471000 -- Propensity to adverse reactions to food
  • 419199007 -- Allergy to substance
  • 416098002 -- Drug allergy
  • 414285001 -- Food allergy
  • 59037007 -- Drug Intolerance
  • 235719002 -- Food Intolerance
name string M This is the name or other description of the product or agent that causes the intolerance.
eventDate DateTime O This is a date that expresses when this particular allergy or intolerance was known to be active for the patient.
reactionFreeText string O This is the reaction that may be caused by the product or agent
codedReaction SimpleCode O

This value is a code describing the reaction Valid values from C80/2.2.3.4.1 are SNOMED CT codes.
severityFreeText String O This is a description of the level of severity of the allergy or intolerance
codedSeverity SimpleCode O

This value is a code describing the level severity of the allergy or intolerance. Valid values are:

  • 255604002 -- Mild
  • 371923003 -- Mild to Moderate
  • 6736007 -- Moderate to severe
  • 24484000 -- Severe
  • 399166001 -- Fatal

4.3. Clinical Requirement

Allergy is characterized by:

  • name of the allergy or intolerance
  • severity of the allergy or intolerance
  • reaction of the allergy or intolerance
